# Turbo Vector

`Turbo Vector` is a GitHub Pages-friendly top-down racing starter, but the real target is a fuller desktop arcade racer in the spirit of *Super Cars*.

The production direction lives in [DESIGN-BIBLE.md](./DESIGN-BIBLE.md).

This first pass is a lightweight playable prototype focused on:

- readable top-down handling
- immediate lap-based replay value
- a fast static deployment path with no build step

Use the browser version to evaluate camera feel, speed readability, and the general arcade tone before a larger systems pass.
